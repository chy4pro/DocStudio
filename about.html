DocStudio 使用指南

欢迎使用 DocStudio！这篇指南将详细介绍所有功能的使用方法。

基础设置

🔧 初始配置

首次使用时，需要配置 API 设置：

  1. 点击界面右上角的⚙️设置图标
  2. 填写以下信息：
    • API 接口地址：完整的 API 端点（如：https://api.openai.com/v1）
    • API Key：您的 API 密钥
    • 模型名称：要使用的模型（如：gpt-3.5-turbo）
    • 温度：影响输出的随机性（0-2，较低值产生更确定的输出）
    • 最大令牌数：限制每次请求的最大字数
  3. 点击"测试连接"确保设置正确
  4. 点击"保存设置"完成配置
所有设置都存储在浏览器本地，确保您的 API 密钥安全。

左侧工作区

📝 基本编辑

左侧是您的主要工作区，您可以：

  • 直接输入或粘贴文本
  • 使用 Markdown 语法格式化文本
  • 内容会自动保存到浏览器本地存储
🔄 自动建议功能

工作区顶部有"AI建议"开关：

  • 开启后，在您停止输入 5 秒后会自动提供建议
  • 建议会添加到文本末尾，以"--- AI建议 ---"标记
  • 如果不需要建议，可以随时关闭此功能
📋 整理功能

使用"整理"按钮优化内容：

  • 点击"整理"开始处理
  • AI 会实时输出优化后的内容
  • 可以随时点击"停止"中断处理
  • 完成后可以点击"回退"恢复原始内容
建议在完成初稿后使用此功能，它会帮助精简和组织您的内容。

右侧显示区

📄 Markdown 预览

右侧区域支持实时 Markdown 预览：

  • 打开"自动渲染"开关启用预览模式
  • 可以直接在预览界面中编辑内容
  • 内容会自动同步到 Markdown 文本
  • 支持完整的 Markdown 语法显示
🔄 生成功能

使用中间的"generate"按钮生成文档：

  • 将左侧工作区的内容转换为结构化文档
  • 生成的文档会显示在右侧
  • 生成过程中会显示进度提示
  • 失败时会显示错误信息
🖼️ HTML 预览

使用"render"按钮预览 HTML：

  • 将当前内容转换为完整的 HTML 页面
  • 在全屏模式下展示渲染结果
  • 可以选择在新标签页中打开
  • 支持关闭预览返回编辑界面

AI 功能

💡 右键菜单功能

在任何编辑区域右键点击可以：

  • 对选中的文本提出问题
  • 获取 AI 的即时回答
  • 回答会插入到当前位置
  • 支持中文输入法
选中特定文本后再右键，AI 会专注于回答关于该文本的问题。

其他功能

💾 自动保存

所有内容都会自动保存：

  • 编辑器内容自动保存到本地
  • API 设置保存到本地
  • 开关状态会被记住
  • 刷新页面不会丢失内容
⌨️ 快捷操作

常用快捷键：

  • Esc - 关闭右键菜单
  • Enter - 在右键菜单中提交问题
注意：为了获得最佳体验，建议使用现代浏览器（Chrome、Firefox、Safari 或 Edge 的最新版本）。
